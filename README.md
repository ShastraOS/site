## Tools Used
- Next.js
- TypeScript
- TailwindCSS
- RadixUI
- Vercel (For Deployment)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dv
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
